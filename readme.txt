Ce plugin analyse les liens contenus dans un article et les affiche dans une metabox dans l'interface d'administration.

== Description ==

[WP Link Analysis](http://www.diije.fr/wp-link-analysis/) vous permet d'analyser d'un seul coup d'oeil les liens présents dans un article.
Ce plugin est idéal pour la modération de blogs multi-auteurs, ou encore de sites de communiqués de presse.

== Installation ==

1. Uploader le répertoire `wp-link-analysis` dans `/wp-content/plugins/`
1. Activer le plugin dans le menu 'Extensions' de WordPress

== Frequently Asked Questions ==

= Quels éléments le plugin relève-t-il dans les articles ? =

Le plugin affichera l'élément href des liens, et l'anchor-text (texte contenu entre [<a href="">] et [</a>]).
